# API Overview

The Mansariya API provides real-time bus tracking, route discovery, journey planning, and admin management for Sri Lanka's crowdsource-powered bus system.

## Base URL

| Environment | URL |
|-------------|-----|
| Production | `https://api.masariya.lk` |
| Local Development | `http://localhost:9900` |

## Authentication

Most endpoints are **public** and require no authentication.

**Admin endpoints** (`/api/v1/admin/*`) require an API key in the `X-API-Key` header:

```bash
curl -H "X-API-Key: your-key" https://api.masariya.lk/api/v1/admin/routes
```

## Response Format

All responses are JSON. Errors follow a consistent format:

```json
{
  "error": "descriptive error message"
}
```

## Interactive Docs

The backend also serves an interactive Swagger UI at:
- Local: [http://localhost:9900/docs](http://localhost:9900/docs)
- Production: [https://api.masariya.lk/docs](https://api.masariya.lk/docs)

## Rate Limits

No rate limits are enforced in the current version. The system is designed for mobile app traffic patterns.
